# PressureMatch

My take on a Candy Crush clone :)

PressureMatch is a browser match-3 survival game. Swap adjacent pieces, make matches, chase cascades, and keep the pressure meter from hitting 100%.

## Live

After GitHub Pages finishes building, the game will be available at:

https://frankstop.github.io/PressureMatch/

## Features

- 8x8 match-3 board
- Valid adjacent swaps only
- Automatic match clearing, gravity, refill, and cascades
- Score, high score, moves, timer, and run stats
- Pressure meter lose condition
- Bright and night modes
- Mobile-friendly layout
- Local high score storage
- Google Analytics tag included

## Run Locally

Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```
